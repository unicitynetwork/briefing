SDK Modernization: Extism to Wasmtime Migration
Migrated Rust SDK from Extism to wasmtime Component Model (#29) and hardened the public API for v0.6.0 release (#30). This foundational shift improves WebAssembly interoperability and stability while introducing breaking changes that consolidate the SDK's external interface for long-term maintainability.

Enhanced WIT Type Serialization & Code Generation
Enabled serde derives and unused type generation for WIT-derived types (#32), improving developer ergonomics by automatically generating serialization implementations. This reduces boilerplate code and enables seamless JSON/binary serialization workflows for WIT-defined data structures.
