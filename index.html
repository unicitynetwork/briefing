Agent Lifecycle & IPC Management Infrastructure
PR #673 introduces management IPC channels for the Astrid kernel and core, enabling structured control over agent lifecycle operations. The implementation adds support for quotas, groups, and capability (caps) management, giving the runtime fine-grained authority over agent resource consumption and permissions. This closes issue #672 and lays foundational plumbing for multi-agent orchestration scenarios where isolation and resource governance are critical. The IPC layer allows external management surfaces to interact with agent state without bypassing kernel-enforced boundaries.
