# Unicity Project — Daily Briefing

Daily GitHub activity briefings for the Unicity project, covering three orgs and all tracked repos.

**Live page:** https://unicitynetwork.github.io/briefing/

## Orgs tracked
- [unicitynetwork](https://github.com/orgs/unicitynetwork/repositories)
- [unicity-sphere](https://github.com/orgs/unicity-sphere/repositories)
- [unicity-astrid](https://github.com/orgs/unicity-astrid/repositories)
- [ristik/ndsmt-experiments](https://github.com/ristik/ndsmt-experiments) (commits via browser)

## Daily workflow
Generate report in Claude → push updated `index.html` → page updates live.

## Sweep method
Each report runs `involves:USERNAME` for all 13 team members to catch closes, reviews, comments and assignments — not just authored items.
